A full-stack web application designed to manage civic complaints and waste-related issues efficiently.

 Features

 User Features
- User registration and login
- Complaint submission with image upload
- Location-based reporting
- Track report status
- Personal dashboard to view and manage complaints
- Community interaction and updates

  Admin Features
- Admin dashboard with analytics
- View complaints by area
- Update complaint status
- User management
- Graphical analytics for performance and trends
- Area-wise complaint tracking
